Help on the Web Awareness Web Client

Prerequisites

It is recommended that you view these pages with a recent browser, capable of showing Cascading Style Sheets (CSS) and frames. (Such as Mozilla. IE will also do, but the reloading will appear less smooth than with Mozilla.)

Before using this page, you should turn off JavaScript. Many pages use JavaScript to "break out" of frames, and this will break the application. JavaScript is also used for showing pop-up windows, which we don't need either...

Usage

The Usage of this application should be pretty straightforward: The frame on the right hand side will continually update, showing a web pages that have been found using information about your visitors.

The pause button allows you to stop the automatic reloading of the page - this will allow you to follow the links in the pages without being disturbed by the automatic updates.
While the reloading is paused, the start button will appear. Clicking will re-start the updates. (If you don't re-start, the page updates will automatically commence after 30 minutes).

The links on the left side include a link to this page, a link to an administrational page as well as links to other web pages that have been found.
Clicking on any of those links will show the content in the right frame and pause reloading.
